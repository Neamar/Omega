Pourquoi ajouter des ressource à votre document ? La raison est simple : car cela permemt d'illustrer votre propos. On dit qu'une image vaut milles mots, cette phrase est vraie.

Pour ce faire, rien de plus simple : rendez-vous dans la section "Ajouter des ressources" de votre interface de correction. Ensuite, envoyez depuis votre ordinateur les ressources nécessaires et celles-ci seront prêtes à être utilisées. Pour les inclure dans le document \LaTeX, suivez les indications de \doc[correcteur/aide_tex]{cette page}.
